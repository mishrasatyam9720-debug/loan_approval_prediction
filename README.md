# 🏦 Loan Approval Prediction

## 📌 Overview
This project predicts whether a loan application will be approved based on applicant information such as income, education, credit history, loan amount, dependents, and other features. The model is trained using Support Vector Machine (SVM).

## 🚀 Features
- Data Cleaning & Preprocessing
- Handling Missing Values
- Label Encoding of Categorical Features
- Train-Test Split
- Model Training using Support Vector Machine (SVM)
- Model Evaluation with Accuracy Score

## 📂 Dataset Features
- Gender
- Married
- Dependents
- Education
- Self Employed
- Applicant Income
- Coapplicant Income
- Loan Amount
- Loan Amount Term
- Credit History
- Property Area
- Loan Status (Target)

## 🛠️ Technologies Used
- Python
- Google Colab
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## 📊 Model
- Algorithm: Support Vector Machine (SVM)
- Problem Type: Classification

## 📈 Results
- Achieved **96% accuracy** on the test dataset.

## ▶️ How to Run
1. Open the notebook in Google Colab.
2. Upload the dataset.
3. Run all cells sequentially.
4. View the prediction results and model accuracy.
